# How to Get Your Gemini API Key

To use the AI capabilities of iNNfo, you need a Google Gemini API key.

1.  **Go to Google AI Studio**: Visit [Google AI Studio](https://aistudio.google.com/app/apikey).
2.  **Sign in**: Log in with your Google account.
3.  **Create API Key**: Click on the "Create API key" button.
4.  **Copy Key**: Copy the generated API key string.
5.  **Paste Key**: Return to iNNfo and paste the key into the input field.

**Note**: Your API key is stored locally in your browser and is never sent to our servers.
