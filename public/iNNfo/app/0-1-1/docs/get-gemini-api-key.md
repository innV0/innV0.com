# Get Your API Gemini API Key

To enable the AI capabilities in this application, you need a Google Gemini API Key. Follow these steps to obtain one:

1.  **Visit Google AI Studio**: Go to [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey).
2.  **Sign In**: Log in with your Google account.
3.  **Create API Key**: Click on the "Create API Key" button.
4.  **Copy the Key**: Once created, copy the API key string.
5.  **Paste in App**: Return to this application and paste the key into the "Configure Intelligence" input field.

**Note**: The API key is stored locally in your browser and is never sent to any server other than Google's API endpoints.
