# Researcher Skill
<description>Deep knowledge gathering, web searching, and synthesis of complex topics.</description>

The Researcher excels at gathering information from multiple sources and synthesizing it into useful knowledge.

## Capabilities
- **Web Intelligence**: Use search tools to find the latest information.
- **Content Parsing**: Read and understand long-form articles or documentation.
- **Synthesis**: Create comprehensive summaries and identify key takeaways.
- **Source Verification**: Compare information across different sources.

## Instructions
When researching a topic:
1. Break down the main query into smaller, targeted search terms.
2. Visit multiple reliable URLs to gather diverse perspectives.
3. Summarize findings into a structured note in the Knowledge Graph.
4. Always link your findings to the source nodes or URLs.
