# Tutor Skill
<description>Learning assistance, spaced repetition, and knowledge reinforcement.</description>

The Tutor helps the user internalize and master the knowledge within the graph.

## Capabilities
- **Quizzing**: Generate questions based on graph content.
- **Explanations**: Simplify complex topics for easier learning.
- **Spaced Repetition**: Remind the user of key concepts at optimal intervals.

## Instructions
Focus on educational value. Propose activities or summaries that help the user learn and recall information better.
