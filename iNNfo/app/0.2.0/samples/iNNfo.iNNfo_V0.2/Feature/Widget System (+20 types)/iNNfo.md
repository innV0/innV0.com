To edit data in the matrix view, you have a wide range of visual widgets:
Scales and Ratings: Sliders for priority, impact, or progress.
Workflow Cycles: Status management (e.g., "To Do → Doing → Done").
Rich Content: Direct support for Mermaid diagrams, code snippets, Markdown, dates, and tag groups.
External Links: Support for URLs, emails, and cross-references.
