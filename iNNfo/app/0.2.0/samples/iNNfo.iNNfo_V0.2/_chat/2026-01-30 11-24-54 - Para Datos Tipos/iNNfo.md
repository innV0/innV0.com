    1. Arquitectura de Datos Basada en Carpetas
    Distribución Natural: iNNfo utiliza la propia estructura de carpetas de tu sistema operativo como sustrato para el modelo. Cada carpeta que contiene un archivo .iNNfo.md se convierte en un nodo vivo (Tipo o Item).
    Local-First & Privacidad: La aplicación es puramente local y accede a tus archivos mediante la File System Access API. Tus datos nunca salen de tu máquina, excepto cuando interactúas con proveedores de IA.
    Formato Abierto: Utiliza Markdown y YAML para los metadatos, lo que lo hace compatible con herramientas como Obsidian, Logseq, VS Code y sistemas de control de versiones como Git.
    2. Modelado Orientado a Metamodelos
    Definición de Dominio: Puedes crear tu propio lenguaje de datos definiendo tipos personalizados (ej: Tarea, Persona, Proyecto) con iconos, colores y descripciones semánticas.
    Relaciones Tipadas: Configura reglas específicas para cómo se pueden conectar los diferentes tipos de datos entre sí.
    Esquemas de Campos: Define qué atributos debe tener cada tipo de objeto, asegurando la consistencia en todo el gráfico de conocimiento.
    3. Vistas Especializadas
    Navigator View: Un explorador que fusiona la navegación de archivos tradicional con la visualización del modelo, mostrando estados e indicadores de metadatos en tiempo real.
    Matrix View: Una potente cuadrícula interactiva diseñada para la gestión masiva de relaciones y atributos.
    Metamodel View: Genera diagramas automáticos que visualizan la arquitectura y el esquema de tu sistema, ayudándote a entender cómo se conectan tus tipos de datos.
    4. IA Residente y Agentes Especializados
    La aplicación incluye "agentes residentes" que entienden el contexto de tu jerarquía de archivos:

    Architect: Te ayuda a diseñar y refinar tus tipos y relaciones.
    Documenter: Genera resúmenes y documentación automática basada en el contenido de tus carpetas.
    Analyst: Identifica patrones y proporciona información basada en los datos de tu gráfico de conocimiento.
    Validator: Asegura que tu modelo cumpla con las reglas definidas en tu metamodelo.
    5. Sistema de Widgets (+20 tipos)
    Para editar los datos en la vista de matriz, cuentas con una amplia gama de widgets visuales:

    Escalas y Ratings: Deslizadores de prioridad, impacto o progreso.
    Ciclos de Workflow: Gestión de estados (ej: "Por hacer → Haciendo → Hecho").
    Contenidos Ricos: Soporte directo para diagramas Mermaid, fragmentos de código, Markdown, fechas y grupos de etiquetas.
    Enlaces Externos: Soporte para URLs, correos y referencias cruzadas.
    6. Infraestructura Técnica y Seguridad
    Seguridad de API: Las claves de IA (como Gemini) se almacenan cifradas localmente mediante la Web Crypto API.
    Performance: Configuración avanzada de code-splitting para cargas rápidas y uso de componentes virtualizados para manejar grandes volúmenes de datos.
    Extensibilidad: Sistema de "Skills" que permite a la IA realizar operaciones en lote y ejecuciones proactivas sobre el modelo.
